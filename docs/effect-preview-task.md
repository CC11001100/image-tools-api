## 工作流程
1. **分析API**: 查看每个页面的API路径和参数
2. **生成效果图**: 使用不同参数组合调用API生成效果图
3. **组织文件**: 将效果图按功能分类存放在 `/frontend/public/examples/` 目录
4. **更新页面**: 修改对应的页面组件，添加效果展示区域
5. **测试验证**: 确保所有效果图正确显示
